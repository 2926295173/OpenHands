# OpenHands API Reference

Welcome to the OpenHands API Reference. This documentation provides details about the REST API endpoints available in OpenHands.

## Authentication

Most API endpoints require authentication using a JWT token. Include the token in the Authorization header:

```
Authorization: Bearer YOUR_JWT_TOKEN
```

## Base URLs

The API is available at the following base URLs:

- Production: `https://app.all-hands.dev`
- Local Development: `http://localhost:3000`

## API Endpoints

The API is organized into the following categories:

- **Conversations**: Manage conversations and interactions with the AI
- **Files**: Access and manipulate files in the workspace
- **Git**: Perform Git operations
- **Settings**: Configure user settings
- **Options**: Retrieve available options for models, agents, etc.

For detailed information about each endpoint, please navigate through the sidebar.
